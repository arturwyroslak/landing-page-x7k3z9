# Landing Page

Prosty responsywny landing page wygenerowany automatycznie.

## Struktura

- index.html — strona główna
- styles.css — podstawowe style (mobile-first)
- script.js — minimalna interakcja (rok, obsługa formularza)

## Uruchomienie

Otwórz index.html w przeglądarce lub użyj GitHub Pages/Netlify do hostowania.
